# Model Selection Policy

## Objective

Maximize value by selecting the lowest-cost model that can reliably
complete the request.

## Preferred Order

1.  Gemma 2:2B (Local)
2.  Gemini Flash Lite
3.  Gemini 3.5 Flash
4.  Gemini 2.5 Pro
5.  Gemini 3.1 Pro / Extended Thinking

Escalate only when capability, context length, or reasoning requirements
justify it.
