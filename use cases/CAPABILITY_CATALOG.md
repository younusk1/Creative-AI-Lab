# Capability Catalog (Version 2)

This catalog is the authoritative specification for Creative AI Lab
capabilities.

## Design Principles

-   Route by capability, not by model.
-   Select the lowest-cost model that satisfies the request.
-   Separate routing, model selection, and execution.
-   Prefer local execution whenever practical.
